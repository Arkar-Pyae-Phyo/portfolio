## ğŸ”§ Customization

### Changing Colors
Update the CSS custom properties in `:root`:
```css
:root {
  --primary: #your-color;
  --accent: #your-accent;
  /* ... other variables */
}
```

### Adding New Sections
1. Add HTML structure following existing patterns
2. Create corresponding CSS classes
3. Add navigation link
4. Implement scroll animations

### Modifying Animations
Adjust AOS attributes:
```html
<div data-aos="fade-up" data-aos-delay="200" data-aos-duration="800">
```
